Gmail (web)

  1. Fill in your details and pick a theme, then click Copy signature.
  2. In Gmail: gear icon, See all settings, General, scroll to Signature, Create new, then paste (Ctrl+V).
  3. Set it for new emails and replies, then Save changes at the bottom.

Apple Mail

  1. Click Download HTML file.
  2. Open Mail > Settings > Signatures, create a new signature, type any placeholder, then quit Mail.
  3. In Finder press Cmd+Shift+G, go to ~/Library/Mail/, open the V* folder, then MailData/Signatures/. Find the newest .mailsignature file.
  4. Open it in TextEdit. Keep the header lines at the top (down to the first blank line) and replace everything below with the contents of your downloaded file.
  5. Save, then Get Info on the file and tick Locked. Reopen Mail.

Outlook (Windows desktop)

  1. Click Download HTML file.
  2. Open File > Options > Mail > Signatures, click New, name it, assign your account, then OK and close Outlook.
  3. In File Explorer paste %APPDATA%\Microsoft\Signatures into the address bar and press Enter. Find the .htm file named after your signature.
  4. Open it in Notepad, select all, delete, paste in the contents of your downloaded file, and save (keep the filename).
  5. Reopen Outlook.
